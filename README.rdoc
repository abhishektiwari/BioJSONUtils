== BioJSONUtils

JSON Endpoint for the various Bio web services such as NCBI, EBI, KEGG etc

== Dependencies

* Biopython
* Flask (http://flask.pocoo.org/)
* Tornado (http://www.tornadoweb.org/)


== Help & Documentation
For more information about installing, running and configuring BioJSONUtils
including a list of known issues and troubleshooting information, 
refer to: http://wiki.github.com/abhishektiwari/BioJSONUtils/
